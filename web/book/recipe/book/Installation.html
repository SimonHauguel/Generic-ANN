Dépendances

  • Python 3.7+, sur les linux modernes, une version fonctionnelle est déjà installée. Sur Windows, vous pouvez l'installer via le MicrosoftStore
  • Flask, à installer via la ligne de commande suivante : python3.7 -m pip install flask
  • Intel® Quartus® 19.1 (non testé sur d'autres versions)
  • Un navigateur web moderne (autre que Internet Explorer)

Dépendances materielles

  • Un FPGA, pour notre cas nous avons travaillé sur la carte DE0-CV
  • Un câble USB <-> UART

Dépendances optionelles

  • Dans le cas où vous souhaitez héberger vous même ce MdBook, suivez la page d'installation de leur documentation Documentation MdBook
  • L'outil de versionnement Git, dans le cas où vous souhaitez cloner directement les sources.

Installation

Vous retrouverez toutes les sources sur le repository GitHub suivant. Vous pouvez le cloner via : git clone https://github.com/SimonHauguel/Generic-ANN.git

Une fois installé, vous trouverez un fichier d'example wmatrix et un dossier web. Dans le dossier web se trouve, les fichiers sources python et les fichiers sources de ce MdBook.
